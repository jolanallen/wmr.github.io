Le Concept

🧠

Centralisation

Répertorie l'ensemble des vecteurs d'attaques web, des classiques aux plus obscurs, directement exploitables.

🔗

Interactif

Pensé pour Obsidian, profitez d'une navigation fluide entre les techniques, les payloads et les outils.

🤝

Communautaire

Le projet est open-source. Tout le monde peut contribuer pour enrichir cette base de connaissance mondiale.

Inspiration : L'Héritage ADMR

WMR est né d'un constat simple : la mindmap ADMR (Active Directory MindMap Recipes) créée par les élèves de l'école 2600 et celle d'Orange Cyberdefense sont des références incontournables pour l'Active Directory.

Sur cette même philosophie, j'ai décidé de créer l'équivalent pour le Pentest Web. Un outil indispensable pour les auditeurs, bug hunters et étudiants.
